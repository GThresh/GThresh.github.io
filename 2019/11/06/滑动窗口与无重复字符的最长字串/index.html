滑动窗口与无重复字符的最长字串

GThresh

2019-11-06

题目描述 ( 来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/longest-substring-without-repeating-characters )

  给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。

示例 1：

输入: “abcabcbb”

输出: 3

解释: 因为无重复字符的最长子串是 “abc”，所以其长度为 3。

示例 2：

输入: “bbbbb”

输出: 1

解释: 因为无重复字符的最长子串是 “b”，所以其长度为 1。

示例 3：

输入: “pwwkew”

输出: 3

解释: 因为无重复字符的最长子串是 “wke”，所以其长度为 3。

请注意，你的答案必须是 子串 的长度，”pwke” 是一个子序列，不是子串。

解题思路：

一、差一点就完美

  今天总算自己独立写出了完整的解法，大概回顾一下解题思路吧：

  • 这道题首先要对字符串进行拆分，拆分字符串最好不要使用subString的方法，所以这里我使用了charAt()方法把字符串拆分为字符串
  • 然后参考之前hashtable高效率的查找（同时这里需要把拆分的字符串与字符串位置记录，使用hashmap真是太合适了！）
  • 查找第二次出现字符串的位置减去第一次出现字符串的位置并做记录，遍历每个字符串，取最大的一次记录
